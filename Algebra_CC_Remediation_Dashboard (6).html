Remediation Groups

3 groups of 3 built on severity: GCS → count of domains Below Target → average domain index. Same daily target, differentiated scaffolding.

How to use these groups
  • Rotation model: All 3 groups receive the same daily learning target, but differ in scaffolding depth, pace, and independent task. Teacher rotates through 12-15 min small-group instruction with each.
  • Why severity, not single-domain: Every cohort student is Below Target on multiple domains. Grouping by single-domain weakness is not productive. Severity grouping lets you modulate scaffolding instead of re-sequencing content.
  • Reassess after Week 1 checkpoint: Students showing Approaching-level work move up. Students continuing to struggle move down.
  • Same content, different depth: Group A gets concrete representations and slower pace. Group C gets faster pace and more application work.

Suggested Daily Rotation (45-50 min block)

IXL Skill Alignment

Cohort-prioritized version of the GA Milestones Algebra IXL skill plan. Domains ranked by cohort % Below Target. "Start Here" tags flag the highest-leverage skills for this group.

How to use
  • Assign by code: Students enter the skill code (e.g. KNV) in IXL's search to jump directly to the skill. Teachers can also assign by code from the analytics dashboard.
  • "Start Here" tag: These are the highest-leverage skills for THIS cohort, identified by cross-referencing domain Below-Target percentages with the IXL skill list. Assign these first.
  • Tier colors: Domains are sorted and colored by cohort priority. Red banner = Tier 1 Critical, amber = Tier 2 High, yellow = Tier 3 Moderate.
  • Daily assignment map: The second table maps specific IXL codes to each of the 15 plan days. Assign the listed codes as homework or in-class follow-up.
